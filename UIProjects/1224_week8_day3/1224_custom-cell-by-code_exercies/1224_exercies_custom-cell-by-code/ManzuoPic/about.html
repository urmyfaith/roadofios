团购上满座，优享好生活

满座网 （www.manzuo.com） 成立于2010年1月，是中国最早也是最大的团购网站。自成立以来，一直倡导"品质团购，聪明消费"的消费主张，并提出了"团购上满座，优享好生活"的品牌理念，成为中国电子商务领域最具代表性的网站和领军企业。

满座网现有注册用户超过1000万，服务覆盖北上广深等全国110个城市，年收入额将近20亿元。
