# 网络数据的请求下载

- 电阻,反映慢,不能通过手指,需要使用触屏笔

### 下载的三种方式

1. 使用字符串的方法进行同步下载
2. 使用NSURLConnection的方法进行同步下载
3. 使用NSURLConnection进行异步下载


| **同步下载** | **异步下载** |
| ------------- | ------------ |
| 同一时间点,只有***一个***线程在工作  | 同一时间点,只有***多个***线程在工作 |
|同一时间点,只能做一件事情|同一时间点,同事做多件事情|

下载信息的时候,不能影响加载UI
